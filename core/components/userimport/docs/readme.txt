--------------------
Extra: UserImport
--------------------
Version: 1.0.0-beta1
Author: bitego <office@bitego.com> (Martin Gartner)
--------------------
 
An extra to easily import users into the MODX user database and batch 
assign MODX user groups and roles. The importer accepts data files in 
CSV format. 

Reqirements:

- MODx 2.3 and above

Feel free to suggest ideas/improvements/bugs on Github:

https://github.com/gadgetto/UserImport/issues
